Nykka (Clone)
This project is an e-commerce website inspired by Nykaa. It includes features for both regular users and admins, allowing for a robust and functional online shopping experience. While the design and functionality are inspired by Nykaa, this project is not a direct clone and includes unique elements and features.

Features
For Admins
View Products: Access a list of all products in the inventory.
Add Products: Add new products to the catalog.
Update Products: Modify details of existing products.
Delete Products: Remove products from the inventory.
View Users: View all registered users.
Delete Users: Remove users from the system.
For Normal Users
Login/Signup: Register and log in to access the website.
Browse Products: Explore products available in the catalog.
Add to Cart: Add products to your shopping cart.
Account: View order history.

Technologies Used
Frontend: React, Redux, CSS, Chakra UI, Bootstrap
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT (JSON Web Tokens)
Others: Axios for API requests, React Router for navigation
